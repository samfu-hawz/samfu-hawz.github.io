¿Qué es "PI"?

"PI" es una aplicación creada para ayudar a personas con problemas de salud nutricional, no como una respuesta a estos problemas, sino como una ayuda para hacer más llevadera esta experiencia que naturalmente es estresante ya sea para un infante, adolecente, adulto o adulto mayor.

Buscamos que nuestra aplicación tenga una interfaz sencilla, que sea fácil de utilizar para personas con o sin experiencia, de manera que sea cual sea tu edad, conocimiento, personalidad, etcétera te sientas bienvenido/a a utilizar nuestra aplicación.
